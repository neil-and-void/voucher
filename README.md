# How To Run Locally
1. `git clone` to some directory 
2. open up terminal and cd into /backend
3. run `npm run dev`
4. open another terminal and cd into /frontend
5. run `ng serve`
6. open browser and go to http://localhost:4200 or whatever port Angular opens up on
